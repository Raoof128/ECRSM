# Security Policy

## Supported versions
This repository is an educational reference. There are no official releases; treat all versions as best-effort.

## Reporting vulnerabilities
If you discover a vulnerability or security concern:
1. Do not open a public issue with exploit details.
2. Email the maintainers privately with a clear description and reproduction steps.
3. We will acknowledge within 5 business days and provide remediation guidance or mitigations as appropriate.

## Scope and safety
- The eBPF programs are read-only and capture metadata only.
- Simulations are synthetic and local. Do not run in production environments.
- Avoid running with elevated privileges on untrusted systems.
